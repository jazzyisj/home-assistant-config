## TAGS USED IN CODE

- #VERIFY - to be verifed
- #TEST - test code
- #TODO - to be completed
- #NEXT - next feature/fix to complete
- #NOTE - important note / significant detail
- #HACK - something hacky done here
- #IDEA - idea for feature/option/function
- #ISSUE - unresolved issue
- #TEMP - temporary option/code
- #STYLE - css styling issues/notes
- #STARTUP - entity triggering automation at startup
- #TEMPLATE - entity triggering automation on template reload
- #DISABLED - disabled code / option
- #BUG - unfixable bug in HA/component/device
- #BUGFIX - workaround for bug
- #MIDNIGHT - shift day change to day reset time
- #TV - TV issues/notes
- #MASS - music assistant issues/notes
- #INFO - info level log tag
- #WARNING - warning level log tag
- #ERROR - error level log tag
- #CRITICAL - critical level log tag
